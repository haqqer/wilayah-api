WILAYAH API
===================
### HOW TO USE

### Cara penginstalan :
1. Pertama clone atau fork repository ini
2. ```git clone https://github.com/haqqer/wilayah-api```
3. Kemudian ```cd wilayah-api```
4. Jalankan, ```yarn install``` atau ```npm install```
5. ```yarn run dev``` atau ```npm run dev```
